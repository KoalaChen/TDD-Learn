## 可擴充的功能
 * 可線上抽換規則(ex.移除任兩筆不同則打95折折扣)
 * 新增規格(全部金額超過一定金額可打折)
 * 可決定規格優先權(ex.將新增的規則放在第一位)
 * 可新增規格針對不合理的項目中止執行(ex.全部金額計算結果為0)
 * 訂單細項允許重複同樣的產品，可能的原因是同樣商品在倉庫A和倉庫B各有一本
 * 也可以新增幾月幾號後規則改變，只要把規則放前面即可
